🚨 IMPORTANT NOTES
================================================================================

1. BACKEND REQUIREMENT:
   Backend MUST be running at http://127.0.0.1:8001
   Make sure FastAPI server is started before running frontend

2. ENVIRONMENT SETUP:
   Copy .env.example to .env if needed
   All default values point to localhost:8001

3. DEPENDENCIES:
   Run "npm install" before first run
   Ensure Node.js 16+ is installed

4. BROWSER CACHE:
   If having issues, clear browser cache and localStorage
   Use Ctrl+Shift+Delete in Chrome/Firefox

5. PORT CONFLICT:
   Frontend runs on port 3000
   Make sure port isn't already in use
   Change in vite.config.js if needed
